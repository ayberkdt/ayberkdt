## Featured Project — 🌕 Lunar Frozen Orbits Simulator (Python)
A high-fidelity lunar orbit propagation and stability analysis codebase (**14k+ LOC**) designed for research-grade experimentation.

**Key capabilities**
- **Lunar gravity:** `GL1800F` with **adaptive spherical-harmonic degree selection** (up to **1800×1800**)  
- **Perturbations:**  
  - **SRP:** Cannonball model  
  - **Albedo:** Lommel–Seeliger reflectance  
  - **Third-body effects:** ephemeris-driven via **NAIF SPICE** kernels  
- **Integrators:** `DOP853` + selected **symplectic** integrators (for structure-preserving long runs)  
- **Workflow:** configuration-driven runs, diagnostics hooks, and stability monitoring for frozen-orbit analysis

Repo: **[Lunar Frozen Orbits — Repository](REPO_LINK)**
